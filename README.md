# نام پروژه: [ سامانه یکپارچه بودجه و تنخواه "سامان"]
**نسخه: v.۳.۰.۰.۲۵**

یک سیستم جامع و هوشمند برای مدیریت فرآیندهای بودجه‌ریزی، تخصیص بودجه، تنخواه گردانی، و کنترل هزینه‌ها در سازمان‌ها. این سامانه با هدف افزایش شفافیت، بهینه‌سازی فرآیندهای مالی و ارائه گزارش‌های دقیق مدیریتی طراحی شده است.

---

## 🌟 قابلیت‌های کلیدی

*   **مدیریت جامع بودجه:**
    *   تعریف دوره‌های بودجه کلان (سالانه، شش‌ماهه و ...).
    *   ایجاد سرفصل‌های بودجه (BudgetItem) برای هر دوره.
    *   تخصیص بودجه (`BudgetAllocation`) از دوره‌ها به سازمان‌ها و واحدهای مختلف.
    *   تخصیص بودجه از سطح سازمان به پروژه‌ها و زیرپروژه‌ها (مراکز هزینه).

*   **سیستم تنخواه گردانی یکپارچه:**
    *   ثبت درخواست تنخواه (`Tankhah`) توسط کاربران.
    *   اتصال هر تنخواه به یک منبع بودجه مشخص برای کنترل دقیق هزینه‌ها.
    *   ثبت فاکتورها (`Factor`) و ردیف‌های هزینه (`FactorItem`) برای هر تنخواه.
    *   محاسبه آنلاین مانده تنخواه و مانده بودجه مرتبط.

*   **گردش کار (Workflow) پویا و قدرتمند:**
    *   تعریف مراحل گردش کار (`WorkflowStage`) سفارشی برای موجودیت‌های مختلف (تنخواه، دستور پرداخت و ...).
    *   مدیریت تاییدکنندگان (`StageApprover`) و اقدامات مجاز (`PostAction`) بر اساس پست‌های سازمانی.
    *   تاریخچه کامل اقدامات (`ApprovalLog`) برای شفافیت و قابلیت حسابرسی.

*   **دستور پرداخت مکانیزه:**
    *   ایجاد خودکار یا دستی دستور پرداخت (`PaymentOrder`) پس از تایید نهایی هزینه‌ها.
    *   گردش کار الکترونیکی برای جمع‌آوری امضاهای لازم.
    *   ثبت اطلاعات پرداخت نهایی و اتصال به سوابق مالی.

*   **ساختار سازمانی انعطاف‌پذیر:**
    *   تعریف سازمان‌ها، واحدها و شعب به صورت سلسله مراتبی.
    *   ایجاد پست‌های سازمانی (`Post`) و انتساب کاربران به آنها (`UserPost`).
    *   کنترل دسترسی دقیق و مبتنی بر نقش (Role-Based Access Control).

*   **داشبوردها و گزارش‌های مدیریتی:**
    *   داشبورد مدیریتی خلاقانه برای نمایش شاخص‌های کلیدی عملکرد (KPIs) به صورت لحظه‌ای.
    *   گزارش جامع و سلسله مراتبی بودجه با قابلیت Drill-Down از سطح کلان تا جزئی‌ترین فاکتورها.
    *   قابلیت دریافت خروجی در فرمت‌های **PDF** و **Excel**.

*   **امنیت و پایداری:**
    *   استفاده از مکانیزم‌های امنیتی استاندارد Django.
    *   لاگ‌برداری کامل از اقدامات کاربران برای حسابرسی.
    *   سیستم کنترل دسترسی دقیق.

## 🛠️ تکنولوژی‌های مورد استفاده

*   **Backend:** Python 3.11+, Django 5.x
*   **Database:** MySQL
*   **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (ES6+), jQuery
*   **Real-time (اختیاری):** Django Channels (برای نوتیفیکیشن‌های زنده)
*   **API:** Django REST Framework (پیشنهادی) یا ویوهای مبتنی بر `JsonResponse`
*   **Asynchronous Tasks (اختیاری):** Celery & Redis (برای پردازش‌های پس‌زمینه مانند ارسال ایمیل/پیامک)
*   **Reporting:** WeasyPrint (برای PDF), openpyxl (برای Excel)

## 🚀 راهنمای نصب و راه‌اندازی

برای راه‌اندازی پروژه به صورت محلی، مراحل زیر را دنبال کنید:

1.  **کلون کردن پروژه:**
    ```bash
    git clone [آدرس ریپازیتوری شما]
    cd [نام پوشه پروژه]
    ```

2.  **ایجاد و فعال‌سازی محیط مجازی (Virtual Environment):**
    ```bash
    python -m venv venv
    # On Windows
    venv\Scripts\activate
    # On macOS/Linux
    source venv/bin/activate
    ```

3.  **نصب وابستگی‌ها:**
    ```bash
    pip install -r requirements.txt
    ```
    *توجه: مطمئن شوید که فایل `requirements.txt` شما شامل تمام پکیج‌های لازم مانند `Django`, `mysqlclient`, `djangorestframework`, `weasyprint`, `openpyxl`, `jdatetime` و ... باشد.*

4.  **تنظیمات محیطی (`.env`):**
    *   یک فایل `.env` در ریشه پروژه ایجاد کرده و متغیرهای محیطی لازم مانند اطلاعات اتصال به دیتابیس، `SECRET_KEY` و ... را در آن قرار دهید.
    *   یک فایل نمونه به نام `.env.example` برای راهنمایی در پروژه قرار داده شده است.
    ```env
    # .env.example
    SECRET_KEY='your-secret-key'
    DEBUG=True
    DB_NAME='your_db_name'
    DB_USER='your_db_user'
    DB_PASSWORD='your_db_password'
    DB_HOST='127.0.0.1'
    DB_PORT='3306'
    ```

5.  **اعمال مایگریشن‌های دیتابیس:**
    ```bash
    python manage.py makemigrations
    python manage.py migrate
    ```

6.  **ایجاد کاربر Superuser:**
    ```bash
    python manage.py createsuperuser
    ```

7.  **جمع‌آوری فایل‌های استاتیک:**
    ```bash
    python manage.py collectstatic
    ```

8.  **اجرای سرور توسعه:**
    ```bash
    python manage.py runserver
    ```
    حالا می‌توانید پروژه را در آدرس `http://127.0.0.1:8000` مشاهده کنید.
