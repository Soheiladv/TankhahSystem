راهنمای ایجاد پروژه در سیستم

نام پروژه

نام پروژه باید یه عبارت کوتاه و معنی‌دار باشه که هدف یا موضوع پروژه رو مشخص کنه. این اسم باید منحصربه‌فرد باشه تا تو لیست پروژه‌ها راحت پیدا بشه. مثلاً اگه پروژه‌ات درباره ساخت یه ساختمونه، می‌تونی بنویسی "ساختمان مرکزی 1404" یا "پروژه بازسازی انبار".

  • حداکثر طول: 100 کاراکتر
  • کاراکترهای مجاز: حروف فارسی، اعداد، فاصله، و خط تیره (-)
  • پیشنهاد: از اسامی تکراری پرهیز کن و یه چیزی انتخاب کن که بعداً راحت یادت بیاد.
مثال: "پروژه توسعه نرم‌افزار مالی" یا "بازسازی سالن شماره 3"

کد پروژه

کد پروژه یه شناسه یکتاست که برای شناسایی سریع پروژه تو سیستم استفاده می‌شه. این کد معمولاً ترکیبی از حروف و اعداده که یه الگوی مشخص داره، مثلاً بر اساس سال، نوع پروژه یا سازمان. هدفش اینه که تو گزارش‌ها و جستجوها سریع پیداش کنی.

  • حداکثر طول: 20 کاراکتر
  • یکتا بودن: باید تو سیستم تکراری نباشه (سیستم اینو چک می‌کنه).
  • فرمت پیشنهادی: [سال]-[نوع]-[شماره]
    • سال: دو رقم آخر سال شمسی (مثل 04 برای 1404)
    • نوع: یه حرف یا کلمه کوتاه برای نوع پروژه (مثل S برای نرم‌افزار، B برای ساختمان)
    • شماره: یه عدد ترتیبی (مثل 001)
مثال: "04-S-001" (اولین پروژه نرم‌افزاری سال 1404) یا "03-B-012" (دوازدهمین پروژه ساختمانی سال 1403)

نکته مهم

هر دو فیلد "نام پروژه" و "کد پروژه" تو فرم اجباری هستن. اگه کد تکراری وارد کنی، سیستم خطا می‌ده و باید یه کد جدید بذاری.
